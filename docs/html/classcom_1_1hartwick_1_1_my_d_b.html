Detailed Description

MySQL Database class.

This MySQL Database class performs some sanity checking on return values and sets various members to contain the insert ID, number of rows, error values etc.

Definition at line 313 of file mysql.class.inc.

getQS ( )

Get an ID for a Query Set (QS)

This method checks the currently registered Query Set's for an unlocked set so we can reuse an existing Query Set instead of having to create a new one. If there are no registered but unused Query Set's request a new Query Set be created, lock it and return the index of that Query Set.
